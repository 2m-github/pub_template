기본 가이드

1. 언어코드

각 언어별로 폴더를 구분

  • 한국어 : /ko_KR/
  • 영어 : /en_US/
  • 중국어 : /zh_CN/
  • 베트남어 : /vi_VN/
  • 몽골어 : /mn_MN/
  • 우즈베크어 : /uz_UZ/
  • 태국어 : /th_TH/
  • 캄보디아(크메르어) : /km_KH/
  • 네팔어 : /ne_NP/
  • 미얀마(버마어) : /my_MM/

2. 폴더구조

  • HTML : /언어코드/*.html
  • CSS : /언어코드/css/*.css
  • JS : /언어코드/js/*.js
  • image : /언어코드/img/*.jpg|png|gif

3. HTML

charset="UTF-8"

상대경로 사용

각 언어별 html태그에 Language Attribute를 반드시 표기


    < html lang="ko-KR"> 
    < html lang="en-US"> 
    < html lang="zh-CN"> 
    < html lang="vi-VN"> 
    < html lang="mn_MN"> 
    < html lang="uz-UZ"> 
    < html lang="th-TH"> 
    < html lang="km-KH"> 
    < html lang="ne-NP"> 
    < html lang="my-MM">

4. CSS

charset="UTF-8"

상대경로 사용

html 기본 폰트 사이즈 - font-size:10px

단위는 %로 적용하는 요소 외에 rem만 사용 (line-height에 사용되는 em 제외)

  • ./css/common.css : 공통요소
  • ./css/swiper.min.css : 슬라이더 관련
  • ./css/style.css : 공통요소 외 모든 스타일

5. JS

공통기능 외 페이지별 기능은 해당 페이지 html 내에 script로 삽입

  • ./js/jquery-3.4.1.min.js : jQuery
  • ./js/swiper.min.js : 슬라이더 플러그인
  • ./js/common.js : 공통기능
